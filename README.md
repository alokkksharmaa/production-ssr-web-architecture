# Production SSR Web Architecture

This project is a production-style web application focused on understanding
Server-Side Rendering (SSR), backend integration, environment configuration,
and performance tradeoffs in modern web applications.

## Goals
- Understand request lifecycle in SSR applications
- Differentiate server-side vs client-side execution
- Work with environment-based configuration
- Analyze performance and caching behavior
- Follow industry-standard project structure

## Tech Stack (planned)
- Next.js
- Node.js
- REST APIs
- Environment Variables
